## 📝 Sobre o Bootcamp

Este repositório foi criado para documentar e organizar **minha jornada de estudos** no **Bootcamp Santander 2026 - AI Java Back-end**. 

O objetivo do bootcamp é aprender do zero como construir sistemas back-end robustos e seguros com **Java**, uma das linguagens mais utilizadas por grandes bancos e empresas no mundo há mais de duas décadas, e dar o próximo passo integrando **Inteligência Artificial** ao desenvolvimento.

Durante este bootcamp, meu foco é:
* ☕ **Dominar os fundamentos** e a sintaxe do **Java 21**.
* 🛡️ **Construir APIs REST seguras** com **Spring Boot** e **Spring Security**.
* 🏛️ **Aplicar princípios de Clean Architecture** para organizar soluções backend profissionais.
* 🔌 **Integrar serviços externos** com **OpenFeign**.
* 🤖 **Usar IA generativa** com **Whisper OpenAI** e **Gemini** para criar um assistente virtual inteligente por voz como projeto final.
